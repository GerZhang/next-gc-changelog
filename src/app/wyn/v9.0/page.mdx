## V9 版本概览 {{ date: '2025-09-25T00:00Z' }}

Wyn 商业智能 V9 版本现已更新 🎉 在当前版本中，增添如下核心功能：

- AI 智能问数能力增强；
- 设计体验优化。

---

![智能问数 2.0](@/images/wyn-chatbi-2.png)

## 智能问数 2.0 {{ type: 'feature', typeText: 'AI' }}

AI 智能问数在上个版本的基础上进行重磅升级，将大语言模型和嵌入模型集成在产品中，结合客户知识库，提供了更加强大的问数能力，将 BI 从“看数据”推向“懂数据”和“用数据”的新阶段。

- **新增智能分析能力。** 现在用户可以直接通过自然语言沟通，获得基于场景的完整分析结果，包括但不限于图表支撑、数据解读、趋势分析以及洞察总结等。
- **新增嵌入模型支持。** 允许用户将企业内部的数据描述、知识文档转化为知识库，在问数阶段提供个性化的分析结果。
- 优化对话体验，与主流 AI 应用程序交互体验保持一致。支持标准对话与高级分析，类似于像 DeepSeek 对话平台的普通对话与深度思考。

---

## 更多的数据库支持 {{ type: 'feature', typeText: '数据库' }}

- 主数据的接入新增 **人大金仓** 数据库（V8R6及以上版本）的支持。
- 审计日志支持 **达梦** 数据库

---

## ETL 功能增强 {{ type: 'feature', typeText: '数据准备' }}

- ETL 配置现已支持 **撤销** 与 **反撤销** 操作。
- ETL 设计器增加快捷键支持，用户可通过键盘快速完成常用操作。

![敏感数据配置加密](@/images/wyn-property-hide.png)

## 敏感数据支持隐藏/加密 {{ type: 'feature', typeText: '安全' }}

针对于安全性较高的场景中，我们提供了用户、组织结构等信息的隐藏/加密功能。开启该配置后，在系统的任何地方后，相关内容都不会显示明文。

---

## 优化设计体验 {{ type: 'optimize', typeText: '设计' }}

我们在设计阶段，持续提优化开发体验。

- 全新的设计门户。过去设计门户的提供的功能入口相对分散，用户在实际工作中，切换频率较为频繁。新的设计门户将「我的分析」与「创建入口」合并，缩短设计路径，提升设计体验。
- 仪表板设计支持关闭图表刷新。在应对大数据量的图表设计时，关闭图表刷新可以有效提升设计效率，避免因为大屏中多图表联动和数据刷新所导致的卡顿。开发者可以针对于特定的图表手动刷新，满足开发测试需求。
- 其他开发体验细节优化，可通过访问 [Wyn V9.0 预览说明](https://gcdn.grapecity.com.cn/showtopic-239774-1-1.html) 了解更多。
